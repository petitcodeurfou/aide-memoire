Aide-mémoire

Étape numéro un

Aller créer un répertoire pour stocker les fichiers.

Lexique

Le terminal offre une interface en ligne de commande où l'utilisateur peut entrer des commandes directement sous forme de texte. Ces commandes sont interprétées par le système d'exploitation pour effectuer des tâches diverses.

Les commandes peuvent être des instructions pour manipuler des fichiers, des dossiers, lancer des programmes, ou effectuer d'autres opérations système. Par exemple, la commande "ls" affiche la liste des fichiers dans un répertoire.

Un argument de commande permet de préciser des options pour la commande. Ex: "-r" pour "rm".

Liste des commandes

  • ls: la commande "ls" affiche la liste des fichiers dans un répertoire.
  • cd: la commande "cd" permet de changer de répertoire.
  • mkdir: la commande "mkdir" permet de créer un répertoire. Ex: mkdir le_nom_du_nouveau_répertoire
  • rm: la commande "rm -r" permet de détruire un répertoire. Ex: rm -r le_nom_du_répertoire
